# Didactic_Modules_PROTOTYPE
A functional Proof of Concept (PoC) prototype of didactic learning modules. It serves as a foundational platform to bridge early STEM education and Industry 5.0 using an ABB IRB 140.
Project Status: PROTOTYPE
Please note: This repository represents a Proof of Concept (PoC) and Prototype.
Because it is a prototype designed to validate the integration between vision systems, PLCs, and robotic kinematics, the project is currently in an ongoing development state. You may encounter unfinished features, unoptimized code, or sections that require further refinement. The primary goal of this repository is to demonstrate core functionality, architectural design, and industrial communication concepts rather than providing a fully polished, production-ready product.

Feedback and contributions for improvement are always welcome!
